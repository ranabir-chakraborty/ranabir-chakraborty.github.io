AgriStat is a web-based statistical tool developed for agricultural researchers and students.

It supports:

* ANOVA (CRD and RBD designs)
* Multiple response variables
* Post-hoc tests (LSD, Tukey HSD, Duncan)
* Excel data upload
* Interactive result tables

This tool simplifies statistical analysis in agricultural research and removes the need for coding knowledge.

 **Live Tool:**
https://ranabir.shinyapps.io/agristat/
