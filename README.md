## Tools

### AgriStat – Agricultural Statistical Tool

AgriStat is a web-based statistical tool designed for agricultural experiments. It enables:

- ANOVA (CRD, RBD)  
- Multiple response analysis  
- Post-hoc tests (LSD, Tukey, Duncan)  
- Excel-based data input  

👉 Live Tool: https://ranabir.shinyapps.io/agristat/
