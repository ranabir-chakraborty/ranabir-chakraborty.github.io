<h3>AgriStat</h3>

<p><strong>Agricultural Statistical Tool</strong></p>

<p>
AgriStat is a web-based tool designed to simplify statistical analysis in agricultural experiments. It supports ANOVA (CRD, RBD), multiple response analysis, and post-hoc tests.
