## 🛠️ Featured Tool

### AgriStat – Agricultural Statistical Tool

AgriStat is a web-based statistical tool designed for agricultural researchers and students. It enables:

- ANOVA (CRD, RBD)  
- Multiple response analysis  
- Post-hoc tests (LSD, Tukey, Duncan)  
- Excel-based data input  

without requiring any coding knowledge.
